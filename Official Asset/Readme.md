Project 1
时间：2026年1月 - 2026年5月

标题：Syneste动捕研究 Syneste Mocap Research

标签：实时动捕游戏 / 动画管线

一句话描述：基于Vicon动捕的管线研究和游戏原型实践

职责：制作人 / 动画TA / 动捕演员

工具：Vicon / Unreal Engine 5 / Maya

重点：领导力 / 动捕资产优化 / 动捕管线 / Livelink

正文1：【简要介绍】团队由Shuyuan Ji带领，由RIT Magic Spell Studio赞助，旨在研究动捕动画和动捕管线，产出研究文档和动捕管线的使用指导，让更多的学生能够受益于动捕动画技术。团队其他成员包含研究员：Wenpeng Liao和3D美术Kaijie Yang。
正文2：【项目重点】我们基于MAGIC Spell Studio的动捕实验室对Vicon动捕的管线做出了一系列研究。包括利用Livelink将Vicon的角色实时重定向到Metahuman骨骼，并在UE场景中做出交互。实现一个简单的交互式游戏Demo。这个Demo在罗切斯特理工的Imagine RIT大会上获得了数百次的观看和近百次的游玩。我们还研究了Vicon动捕资产转换成可用生产资产的管线，这包含了通过配置文件和脚本在Vicon中优化录制质量，在Maya中利用曲线编辑器进行抖动的修复等等。

网站链接：https://www.artstation.com/artwork/x3gBEY





Project 2
时间：2024年3月 - 2024年5月

标题：核战后50年 50YearsAfterNuclearWar

标签：3D美术 / 概念设计

一句话描述：沉浸式可交互的UE5概念美术项目

职责：全部（单人项目）

工具：UE5 / Maya / Zbrush / Marvelous Designer / Substance Painter

重点：多DCC工具使用 

正文1：【背景介绍】作为这个项目中唯一的设计师和美术，我独立完成了主角的建模与贴图制作，也完成了场景中部分道具的建模与贴图。

同时，我使用了一些 Unreal Marketplace 和 Megascan 资源，例如地铁、地面上的垃圾和碎块、货架、管道等，用来丰富场景细节，比如地面的杂物，或场景中的其他道具。我设计了玩家交互任务流程，并使用蓝图为所有可交互物体制作了逻辑。从打开灯光、寻找撬棍，到穿过车厢并最终逃离，这些元素为玩家提供了一个完整的游戏体验。这是我第一次在一个项目中尝试使用这么多不同的软件。过程很痛苦，但也很有趣。

正文2：【项目重点】这个项目完整的让我学习了从概念 - 角色模型制作 - 脸部雕刻 - 衣服/道具制作 - 材质制作 - 导入游戏的完整流程。在这个过程中，我解决了很多3D美术会遇到的问题，比如UV的拆分，材质的混合等等。整个项目夯实了我作为3D美术的基础。更详细的制作流程可以在网站中看到。
